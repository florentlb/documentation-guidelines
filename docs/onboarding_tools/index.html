Documentation tools

Install the following tools on your machine:

ToolDescription
SlackMost interesting conversations happen on Slack :)
Chrome or Firefox
oXygen XML Editor 20oXygen is the preferred XML editing tool.
Microsoft Office
Git
TortoiseGITUnless you are familiar with using Git from the command line, install a Git GUI, like TortoiseGIT.
Snagit 10Snagit is the screenshot tool we use. You can find the license here
OpenVPNOpenVPN allows you to connect to Talend network remotely. It uses Okta credentials.
Java JDK 1.8Make sure that the Java environment variables are correctly set on your machine after installing Java. The JAVA_HOME variable must exist and point to the installation folder of JAva on your machine. %JAVA_HOME%\bin must exist in the Path variable as well.
